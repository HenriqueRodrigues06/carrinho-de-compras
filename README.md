# 🛒 Carrinho de Compras
 
Projeto simples de carrinho de compras desenvolvido com HTML, CSS e JavaScript puro.
 
## 📋 Sobre o projeto
 
Aplicação web que simula um carrinho de compras, permitindo selecionar produtos de uma lista, informar a quantidade desejada e visualizar o total da compra em tempo real.
 
## ✨ Funcionalidades
 
- Seleção de produtos disponíveis (Fone de ouvido, Celular, Oculus VR)
- Definição de quantidade por produto
- Adição de itens ao carrinho com cálculo automático do subtotal
- Exibição do total geral atualizado a cada adição
- Limpeza completa do carrinho com um clique

## 🚀 Como rodar
 
1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` diretamente no navegador
 
Não é necessário instalar dependências ou configurar servidor.
